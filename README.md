# STM32_lab

嵌入式系統的 STM32 實作與程式碼整理。

## 開發環境
- Board: STM32F407G-DISC1
- MCU: STM32F407VGT6U
- IDE: Keil uVision (MDK-ARM)
- Tool: STM32CubeMX
- Debugger / Programmer: ST-LINK

## Lab Topics
- GPIO
  - Polling
  - Interrupt
- UART
  - Polling
  - Interrupt
- ADC
- Timer
- I2C
- SPI
- Watchdog
- DMA

觀念整理與筆記主要記錄於 HackMD，本 repository 主要用來存放實作程式碼與專案檔。
